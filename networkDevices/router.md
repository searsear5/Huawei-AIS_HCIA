Router คืออะไร?
    Router คืออุปกรณ์ที่ทำงานใน OSI Layer 3 (Network Layer)
    หน้าที่คือ เชื่อมต่อหลาย Network เข้าด้วยกัน และเลือกเส้นทางส่ง Packet โดยใช้ IP Address
    สั้น ๆ:
    Switch ดู MAC (Layer 2)
    Router ดู IP (Layer 3)

เวลา PC จะออก Internet:
    -PC เช็คว่า IP ปลายทางอยู่ subnet เดียวกันไหม
    -ถ้าไม่อยู่ → ส่งไปที่ Default Gateway
    -Default Gateway ก็คือ Router
    -Router จะดู Routing Table
    -ตัดสินใจส่งต่อไป network ถัดไป



    PC1 → Switch → Router → ISP(Internet Service Provider) → Internet → Google


ISP คืออะไร?
ISP (Internet Service Provider) คือบริษัทที่ให้บริการเชื่อมต่อเราเข้าสู่ Internet AIS,TRUE,3BB
